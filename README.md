## ❌ Troubleshooting Common Errors

If the installer fails, try these methods to fix the issue.

### The installation stops moving
Sometimes, security software blocks the installer. Temporarily turn off your antivirus program. Try to install the software again. Turn your antivirus back on after the installation completes.

### Permissions error
Ensure your user account has manager status. If you share this computer, ask the owner to provide access. Run the installer again using the right-click "Run as administrator" option.

### Windows 11 compatibility
Windows 11 includes strict security tools. If the system prevents the program from starting, open the Properties menu for the installer. Go to the Compatibility tab. Check the box labeled "Run this program in compatibility mode for" and choose Windows 10. Apply the change and restart the installer.

### Damaged download file
A corrupted file causes many setup errors. Delete the file you downloaded. Clear your browser cache. Return to the download link and save a fresh copy of the installer.

## 🚀 Setting Up Your First Macro

After you open the program, follow these steps to build your first automation:

1.  Open the Macro Express Pro window.
2.  Click the "New Macro" button in the toolbar.
3.  Choose a trigger. A trigger is the action that starts the automation. Examples include a keyboard shortcut or a mouse click.
4.  Enter the name of your macro.
5.  Click the "Scripting Editor" button to list the steps for the macro.
6.  Add commands such as "Text Type" or "Mouse Move" to the script.
7.  Save the macro and close the editor.
8.  Test the macro by using the trigger you selected.
